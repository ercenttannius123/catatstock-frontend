# StokKu React

Converted from an HTML prototype to a minimal React + Vite app.

Quick start:

1. Open terminal in `c:\Users\winso\Downloads\stokku-react`
2. Install dependencies:

```powershell
npm install
```

3. Run dev server:

```powershell
npm run dev
```

This project includes `chart.js` for charts. The UI is ported into `src/App.jsx` and styles live in `src/index.css`.
